在主菜单里点击多人模式

在直接连接/添加服务器中输入下列地址

```text
cd-2.frp.one:51663
```

![Desktop View](/2025/11/02/7hZEB6.png)

## 服务器设置

进入服务器会发现你并移动不了

几秒后会显示以下信息

![Desktop View](/2025/11/04/7BkSWC.png)

首先按键盘`T`键进入聊天栏

使用鼠标左键点击`同意`绿色字体以进入
服务器点歌房间

(推荐加入 不然每次进入服务器都会提示)

随后输入
```text
/register 公共代码 你的密码 确认密码
```

# 公共代码`Global Password`
我提供的代码 一周-三周更新一次 更新后会通知

# 你的密码`Password`
你的账号密码 以后登录需要使用这个密码

# 确认密码`Password`
你的确认密码 同上条

## 成功进入

随后的设置可以在`ESC`键的设置里调整
包括但不限于`画质/光影/材质包`

在享受整个服务器提供的服务之外
请看注意事项

## 注意事项
规则
:
这个服务器并没有多少规则

但也不代表没有

以下则是你需要注意的东西

1. 没有领地/锁箱/防盗系统 请你自觉
2. 不要辱骂/霸凌/针对他人 请你自觉
3. 禁止开挂/盗刷/滥用行为 请你自觉

> 服务器带有追踪系统 包括但不限于`容器物品/方块破坏放置/杀害实体历史记录'
{: .prompt-danger }
> 一切上述规则一旦违反 你将受到的处罚包括但不限于'封禁IP/公共代码禁止提供/清空背包/公开IP所在地地址` 进入服务器后默认接受规则以及处罚
{: .prompt-danger }
