## 客户端下载&启动器设置

首先下载客户端压缩包

![Desktop View](/2025/11/02/7hOV8Q.png)

## 进入服务器

在主菜单里点击多人模式

在直接连接/添加服务器中输入下列地址

```text
cd-2.frp.one:51663
```

![Desktop View](/2025/11/02/7hZEB6.png)

随后即可进入服务器

## 服务器设置

进入服务器会发现你并移动不了

几秒后会显示以下信息

![Desktop View](/2025/11/04/7BkSWC.png)

首先按键盘`T`键进入聊天栏

使用鼠标左键点击`同意`绿色字体以进入
服务器点歌房间

_忽视的话则会在你每次进入服务器提示 直到点击`同意`_

随后输入
```text
/register 全局代码 你的密码 确认密码
```

# 全局代码`Global Password`
一周至三周更新一次 更新后会重置注册状态 需要重新执行一遍注册指令
_联系我获取全局代码_

# 你的密码`Password`
你的账号密码 以后登录需要使用这个密码

# 确认密码`Password`
你的确认密码 同上条

请看注意事项

## 注意事项

规则
: 这个服务器并没有多少规则

但也不代表没有

以下则是你需要注意的东西

1. 没有领地/锁箱/防盗系统 请你自觉
2. 不要辱骂/霸凌/针对他人 请你自觉
3. 禁止开挂/盗刷/滥用行为 请你自觉

> 服务器带有追踪系统 包括但不限于
>
> `容器移除添加物品/方块破坏放置/杀害生成实体/区块更新历史记录`
{: .prompt-warning }
> 
> 一切上述规则一旦违反 你将受到的处罚包括但不限于
>
> `封禁IP/公共代码禁止提供/清空Java与基岩双端存档/公开IP所在地地址`
>
> 进入服务器后默认接受规则以及违反后处罚
{: .prompt-danger }

登录
: 后续每次进入都可能需要重新登录

其显示为

![Desktop View](/2025/11/04/7B6AOM.png)

首先按键盘`T`键进入聊天栏

随后输入
```text
/l 你的注册密码
```
即可登录

基岩互通
: 此服务器支持`Java/基岩`互通

互通的内容包括但不限于`背包物品/末影箱物品/经验值/最后退出位置`

> _绑定互通后 其账号存档内容跟随Java端 基岩端存档需要取消绑定后才可访问_
{: .prompt-warning }

互通教程
