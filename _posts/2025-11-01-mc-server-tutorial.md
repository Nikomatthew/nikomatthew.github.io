## 设置&附加事项

### 白名单获取

正常来说 进入服务器之前需要白名单

如果没有 则会显示下图

![白名单](/2025/11/14/CZ2GXO.jpeg)

在申请白名单前 请阅读

`注意事项`→`规则`

随后将下列文字发送至我的`邮件`/`QQ`

```text
ID:你的ID
客户端:Java/Bedrock
```

联系方式均可在博客`电脑端左下角`/`手机端选项列表`找到

#### 申请相关问题
ID:不可为中文

客户端:单端只写一个 双端就都写上

邮箱:已废弃 因为自动邮箱发送服务太贵了

进入服务器则认定你已阅读并同意规则及处罚


### 服务器设置

进入服务器会发现你并移动不了

几秒后会显示以下信息

![注册](/2025/11/04/7BkSWC.png)

首先按键盘`T/聊天栏`键进入聊天栏

使用`鼠标左键/触摸`点击`同意`绿色字体以进入
服务器点歌房间

> _忽视的话则会在你每次进入服务器提示 直到点击`同意` 如果没有显示可无视该条_

随后输入
```text
/register 你的密码 确认密码
```

### 皮肤设置

此服务器旧版使用`Littleskin`

新版修改为了服务器端模组

可以去各个皮肤网站搜索皮肤并获取raw链接以让服务器获取皮肤png图片

以`Littleskin`网站皮肤为例

<https://littleskin.cn/skinlib/show/574023>
```text
https://littleskin.cn/skinlib/show/574023
```

只需要简单的将该网站的链接改为

<https://littleskin.cn/raw/574023>
```text
https://littleskin.cn/raw/574023
```

再按`T/聊天栏`键

输入以下指令即可

```text
/skin set web classic "https://littleskin.cn/raw/574023"
```

其他皮肤同理 

只需要将指令里的id改为你需要的皮肤即可

#### 皮肤注意事项(基岩端必看)

因为基岩端的不可抗因素

进入后会自动获取你的`Xbox`皮肤导致你的皮肤被覆盖

只需要输入该指令即可(每次登录都需要)

```text
/skin refresh
```

#### 规则

这个服务器并没有多少规则

但也不代表没有

以下则是你需要注意的东西

1. 没有领地/锁箱/防盗系统 请你自觉
2. 不要辱骂/霸凌/针对他人 请你自觉
3. 禁止开挂/盗刷/滥用行为 请你自觉

> 服务器带有追踪系统 包括但不限于
>
> `容器移除添加物品/方块破坏放置/杀害生成实体/区块更新历史记录`

> 一切上述规则一旦违反 你将受到的处罚包括但不限于
>
> `封禁/白名单删除/清空Java与基岩双端存档`
>
> 进入服务器后默认接受规则以及违反后处罚
{: .prompt-danger }

#### 登录

后续每次进入都可能需要重新登录

其显示为

![登录](/2025/11/04/7B6AOM.png)

首先按键盘`T`键进入聊天栏

随后输入
```text
/l 你的注册密码
```
即可登录


#### 基岩互通

此服务器支持`Java/基岩`互通

互通的内容包括但不限于`背包物品/末影箱物品/经验值/最后退出位置`

> 绑定互通后 其账号存档内容跟随Java端 基岩端存档需要取消绑定后才可访问
{: .prompt-warning }

首先登录Java

随后输入以下指令
```text
/linkaccount .基岩账号名
```
> 注意 基岩账号名前面的点一定要加上 因为防止名字冲突 基岩端进入后名字会自动判定为`.你的基岩账号名`
{: .prompt-warning }
 
![登录](/2025/11/16/Cjom1m.png)

显示该图文字后记录下你的指令

![实例](/2025/11/16/Cjo3eO.png)

退出Java 随后进入基岩端

输入上条指令即可 随后退出重进

重新进入后 你的存档应该跟随Java端

如果没有 请重新操作

取消绑定请输入下列指令
```text
/unlinkaccount
```
绑定后 基岩端与Java端不能同时登录
